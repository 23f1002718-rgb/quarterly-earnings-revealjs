Risk-Adjusted Return on Capital (RAROC)

We evaluate business lines using a risk-adjusted metric:

\[ \text{RAROC} = \frac{\text{Expected Income} - \text{Expected Losses}} {\text{Economic Capital}} \]

Example: \[ \text{RAROC} = \frac{120\text{M} - 20\text{M}}{500\text{M}} = 20\% \]

Explain that RAROC normalizes returns by the economic capital consumed. Walk through the example: 120M income, 20M expected losses, 500M capital, giving 20% RAROC. Stress that we compare this with hurdle rates to decide capital allocation.

Computing Metrics Programmatically


def raroc(expected_income, expected_losses, economic_capital):
    """
    Compute Risk-Adjusted Return on Capital (RAROC).
    All inputs are assumed in the same currency units.
    """
    net_income = expected_income - expected_losses
    return net_income / economic_capital

# Example:
raroc_value = raroc(120e6, 20e6, 500e6)
print(f"RAROC: {raroc_value:.2%}")
